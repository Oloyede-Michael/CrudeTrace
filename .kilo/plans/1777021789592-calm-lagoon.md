# CrudeTrace Landing Page Upgrade Plan

## Current State Analysis
The existing landing page consists of:
- Navbar with logo, nav links, and dashboard button
- Hero section with animated canvas background, headline, subheadline, and CTA buttons
- Features section with three capability cards (Theft Detection, Live Dashboard, Automated Settlement)

## Upgrade Goals
Transform the landing page into a world-class, enterprise-grade SaaS website for the oil and gas supply chain industry while preserving existing sections and content. The redesign should:
- Convey trust, security, and enterprise reliability
- Target governments, petroleum corporations, logistics firms, regulators, and investors
- Incorporate modern, futuristic, industrial design with premium feel
- Use specified color palette and typography
- Implement smooth animations and microinteractions
- Add all requested premium sections
- Maintain responsiveness and performance

## Current Progress

## Technical Implementation

### Stack
- Vite (existing setup)
- Tailwind CSS for styling
- Framer Motion for animations
- TypeScript for type safety
- React 18
- react-router-dom for routing (existing)

Note: The user has chosen to keep the current Vite setup rather than migrating to Next.js. We will upgrade the landing page within the existing Vite project while preserving all existing functionality.

### Performance Optimization
- Image optimization using Next.js Image
- Lazy loading for below-fold content
- CSS optimization and purging
- Minimize JavaScript bundle size
- Implement loading skeletons
- Optimize animations for 60fps

### SEO & Accessibility
- Semantic HTML structure
- Proper heading hierarchy
- Alt text for all images
- ARIA labels for interactive elements
- Sufficient color contrast (WCAG AA)
- Schema markup for organization, FAQ, etc.
- Meta tags for social sharing
- XML sitemap and robots.txt

### Animation Details (Framer Motion)
- Hero: parallax world map, floating nodes, pulsating dots
- Sections: fade-up reveals on scroll
- Cards: hover lift, glow effects
- Buttons: microinteractions (press, hover)
- Counters: number animation on viewport enter
- Ticker: smooth horizontal movement
- Accordions: smooth height transition

### Responsive Design
- Mobile-first approach
- Breakpoints: 640px, 768px, 1024px, 1280px
- Hamburger menu for mobile navbar
- Stack columns appropriately
- Adjust padding and spacing for mobile
- Ensure touch targets are adequate
